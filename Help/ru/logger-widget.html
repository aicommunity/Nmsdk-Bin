Обзор

Виджет журнала отображает сообщения журнала и системную информацию из приложения NMSDK. Он предоставляет обратную связь в реальном времени о событиях приложения, ошибках, предупреждениях и отладочной информации.

Доступ к виджету

Виджет журнала отображается в док-панели главного окна приложения. Вы можете показать или скрыть его с помощью пункта меню Окно → Журнал.

Возможности

  • Журналирование в реальном времени: Отображает сообщения журнала по мере их появления
  • Несколько уровней журнала: Показывает информацию, предупреждения, ошибки и отладочные сообщения
  • Автоматические обновления: Постоянно обновляется новыми записями журнала
  • Прокручиваемое представление: Навигация по истории журнала
  • Текстовое отображение: Формат обычного текста для удобного чтения

Использование

Просмотр журналов

Журнал автоматически отображает сообщения из приложения. Записи журнала включают:

  • Информацию о времени
  • Уровень журнала (Информация, Предупреждение, Ошибка, Отладка)
  • Источник сообщения
  • Содержимое сообщения

Уровни журнала

  • Информация: Общие информационные сообщения
  • Предупреждение: Предупреждающие сообщения о потенциальных проблемах
  • Ошибка: Сообщения об ошибках, указывающие на проблемы
  • Отладка: Подробная отладочная информация (когда включен режим отладки)

Навигация по журналам

  • Используйте полосу прокрутки для навигации по истории журнала
  • Журнал автоматически прокручивается для показа последних сообщений
  • Вы можете прокрутить вверх для просмотра более старых сообщений
Совет: Журнал полезен для отладки проблем и мониторинга поведения приложения. Регулярно проверяйте его, чтобы быть в курсе системных событий.

Конфигурация журнала

Уровни журнала и вывод можно настроить через:

  • Панель статуса - Включение/выключение отладочных сообщений и подробного журналирования
  • Настройки приложения
  • Параметры командной строки

Связанные виджеты

  • Панель статуса - Управление опциями журналирования
  • Окно наблюдения - Мониторинг данных компонентов
