Обзор

Виджет профилирования отображает информацию о производительности времени выполнения компонентов в вашей модели нейронной сети. Он помогает выявлять узкие места и оптимизировать производительность модели, показывая, сколько времени занимает выполнение каждого компонента.

Доступ к виджету

Виджет профилирования можно открыть из пункта меню Окно → Профилирование. Он появляется в док-панели, показывающей таблицу времени выполнения.

Возможности

  • Отображение времени выполнения: Показывает время выполнения для отдельных компонентов
  • Выбор компонентов: Добавление конкретных компонентов в таблицу профилирования
  • Группы компонентов: Добавление всех подкомпонентов выбранного компонента
  • Базовые компоненты: Всегда отображает базовые компоненты (Model, Overhead, Full Step, Gui)
  • Обновления в реальном времени: Время выполнения обновляется во время выполнения модели
  • Анализ производительности: Выявление медленных компонентов и возможностей оптимизации

Использование

Просмотр базовых компонентов

Таблица профилирования всегда отображает базовые метрики производительности:

  • Model: Общее время выполнения модели
  • Overhead: Время системных накладных расходов
  • Full Step: Время полного шага вычисления
  • Gui: Время обновления GUI

Добавление компонентов для профилирования

  1. Нажмите Выбрать компонент
  2. Перейдите к компоненту, который хотите профилировать
  3. Компонент будет добавлен в таблицу профилирования
  4. Время выполнения этого компонента будет отображаться во время выполнения модели

Добавление групп компонентов

  1. Нажмите Добавить группу компонентов
  2. Выберите компонент, содержащий подкомпоненты
  3. Все подкомпоненты будут добавлены в таблицу профилирования
  4. Это полезно для анализа иерархических структур компонентов

Добавление всех компонентов

Нажмите Выбрать все компоненты для добавления всех внешних (не вложенных) компонентов в таблицу профилирования.

Удаление компонентов

  • Нажмите Удалить все для удаления всех небазовых компонентов из таблицы
  • Базовые компоненты (Model, Overhead, Full Step, Gui) нельзя удалить

Интерпретация результатов

Таблица профилирования показывает:

  • Имя компонента
  • Время выполнения на шаг (в миллисекундах или микросекундах)
  • Процент от общего времени выполнения

Используйте эту информацию для:

  • Выявления компонентов, занимающих больше всего времени
  • Поиска возможностей оптимизации
  • Балансировки вычислительной нагрузки
  • Проверки того, что компоненты выполняются как ожидается
Совет: Сосредоточьтесь на компонентах с высоким временем выполнения. Это лучшие кандидаты для оптимизации.

Оптимизация производительности

При анализе данных профилирования:

  1. Ищите компоненты с неожиданно высоким временем выполнения
  2. Проверьте, показывают ли группы компонентов неравномерное распределение вычислений
  3. Рассмотрите возможность оптимизации или замены медленных компонентов
  4. Убедитесь, что накладные расходы разумны по сравнению со временем выполнения модели

Связанные виджеты

  • Панель статуса - Общая статистика выполнения
  • Виджет графика - Визуализация трендов производительности
  • Окно наблюдения - Мониторинг поведения компонентов
