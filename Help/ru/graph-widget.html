Обзор

Виджет графика предоставляет визуализацию числовых данных из вашей модели нейронной сети в реальном времени. Он отображает временные графики, показывающие, как свойства компонентов изменяются со временем во время выполнения модели.

Доступ к виджету

Виджет графика можно открыть из меню Окно. Он появляется в отдельном окне или док-панели, позволяя одновременно мониторить несколько серий данных.

Возможности

  • Несколько серий: Отображение нескольких серий данных на одном графике
  • Обновления в реальном времени: Графики автоматически обновляются по мере поступления данных
  • Настраиваемые оси: Установка пользовательских меток и диапазонов для осей X и Y
  • Цветовое кодирование: Каждая серия может иметь свой цвет
  • Выбор источника данных: Выбор свойств компонентов для мониторинга
  • Автомасштабирование: Автоматическое или ручное масштабирование осей
  • Поддержка матриц: Отображение матричных данных с индексами X и Y

Использование

Добавление серии данных

  1. Нажмите кнопку Выбрать выход
  2. Перейдите к компоненту и свойству, которое хотите мониторить
  3. Для матричных свойств укажите индексы X и Y
  4. Выберите канал вычислений (если существует несколько каналов)
  5. График начнет отображать данные из выбранного свойства

Настройка параметров графика

  • Настройки графика: Нажмите кнопку настроек для конфигурации меток осей, диапазонов и опций отображения
  • Выбор цвета: Изменение цвета отдельных серий графика
  • Имя серии: Каждая серия может иметь пользовательское имя для идентификации

Макет графика

Виджет графика поддерживает:

  • Пользовательские метки осей X и Y
  • Ручные или автоматические пределы осей
  • Отображение последних N элементов (скользящее окно)
  • Автообновление правой границы (для временных рядов)

Удаление серий

  • Выберите серию из списка
  • Нажмите Удалить текущий элемент для его удаления
  • Используйте Удалить все для очистки всех серий

Источники данных

Виджет графика может мониторить:

  • Скалярные свойства (одиночные значения)
  • Матричные свойства (с индексами X и Y)
  • Свойства из любого канала вычислений
  • Несколько свойств одновременно

Режимы отображения графика

  • Непрерывное обновление: График обновляется в реальном времени по мере поступления новых данных
  • Скользящее окно: Отображение только последних N точек данных
  • Фиксированный диапазон: Отображение данных в фиксированном временном или значном диапазоне
Совет: Используйте несколько виджетов графика для мониторинга различных аспектов вашей модели. Каждый виджет может отображать различные свойства или использовать различные настройки отображения.

Связанные виджеты

  • Окно наблюдения - Расширенный мониторинг с несколькими типами графиков
  • Виджет изображений - Визуализация данных изображений
  • Виджет профилирования - Анализ производительности
