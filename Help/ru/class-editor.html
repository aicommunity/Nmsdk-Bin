Обзор

Редактор классов позволяет просматривать и редактировать свойства классов компонентов в системе. Он предоставляет интерфейс для изучения определений классов, их свойств, методов и документации.

Доступ к виджету

Редактор классов можно открыть из пункта меню Окно → Описание классов. Он появляется в отдельном окне.

Возможности

  • Выбор классов: Просмотр и выбор классов компонентов из списка
  • Отображение информации о классах: Просмотр подробной информации о выбранных классах
  • Просмотр свойств: Просмотр всех свойств, методов и атрибутов класса
  • Документация: Просмотр документации и описаний классов
  • Иерархия классов: Понимание наследования и отношений классов

Использование

Просмотр информации о классах

  1. Откройте редактор классов
  2. Просмотрите список классов слева
  3. Выберите класс для просмотра его информации
  4. Детали класса будут отображены справа

Список классов

Левая панель показывает иерархический список всех доступных классов компонентов:

  • Навигация по категориям для поиска классов
  • Щелкните на классе для его выбора
  • Информация о выбранном классе появляется в панели отображения

Отображение информации о классах

Правая панель отображает подробную информацию о выбранном классе:

  • Имя и описание класса
  • Иерархия наследования
  • Свойства и их типы
  • Методы и функции
  • Документация и примечания по использованию

Свойства классов

Для каждого класса вы можете просмотреть:

  • Параметры: Настраиваемые параметры класса
  • Входы: Входные свойства, которые принимает класс
  • Выходы: Выходные свойства, которые генерирует класс
  • Переменные состояния: Внутренние переменные состояния
  • Методы: Доступные методы и функции
Совет: Используйте редактор классов для понимания возможностей компонентов перед добавлением их в модель или для изучения доступных свойств и методов.

Связанные виджеты

  • Виджет списка классов - Быстрый браузер классов в движке рисования
  • Виджет списка компонентов - Просмотр экземпляров классов в модели
  • Работа с компонентами - Общая информация о компонентах
