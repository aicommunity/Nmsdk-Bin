Создание нового проекта

  1. Выберите Файл → Создать конфигурацию из меню
  2. Выберите расположение и имя для вашего проекта
  3. Настройте параметры проекта по необходимости
  4. Нажмите OK для создания проекта

Добавление компонентов

  1. Откройте окно Управление компонентами через Окно → Управление компонентами
  2. Просмотрите или найдите нужный компонент
  3. Дважды щелкните по компоненту, чтобы добавить его в модель
  4. Компоненты появятся в представлении Modern Diagram

Соединение компонентов

  1. Откройте вкладку Modern Diagram в окне Draw Engine
  2. Щелкните на выходном порту (синий кружок справа от узла)
  3. Перетащите на входной порт (зеленый кружок слева от другого узла)
  4. Отпустите для создания соединения
Совет: Наведите курсор на порты, чтобы увидеть подробную информацию в подсказках, включая имена портов, типы и доступные действия.

Настройка компонентов

  1. Выберите компонент в окне управления компонентами
  2. Используйте редактор свойств для изменения параметров компонента
  3. Изменения применяются немедленно

Запуск модели

  1. Используйте меню Вычисление для управления выполнением модели
  2. Нажмите Старт для начала выполнения
  3. Используйте Пауза для временной остановки выполнения
  4. Используйте Сброс для сброса состояний компонентов

Мониторинг результатов

  1. Откройте Окно → Окно наблюдения для создания окон мониторинга
  2. Добавьте серии для наблюдения за конкретными свойствами компонентов
  3. Графики будут обновляться в реальном времени во время выполнения
