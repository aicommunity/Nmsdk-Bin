Обзор

Виджет движка рисования — это главный визуальный интерфейс для просмотра и редактирования вашей модели нейронной сети. Он содержит визуализацию схемы компонентов вместе со списком доступных классов компонентов. Виджет предоставляет два режима визуализации: устаревшее представление диаграммы и современное представление диаграммы.

Доступ к виджету

Виджет движка рисования отображается в главном окне приложения, обычно во вкладке или подокне. Он автоматически создается при запуске приложения и является основным интерфейсом для работы со структурой вашей модели.

Возможности

  • Двойной режим визуализации: Переключение между устаревшим и современным представлениями диаграммы
  • Список классов компонентов: Просмотр и поиск доступных классов компонентов
  • Выбор компонентов: Щелчок по компонентам для выбора и просмотра их свойств
  • Навигация: Переход во вложенные компоненты двойным щелчком
  • Обновления схемы: Автоматические обновления при изменении компонентов или соединений

Компоненты

Список классов компонентов

Левая панель отображает иерархический список доступных классов компонентов. Вы можете:

  • Просматривать категории компонентов
  • Искать конкретные компоненты
  • Дважды щелкать для добавления компонентов в модель

Устаревшее представление диаграммы (UDrawEngineImageWidget)

Устаревшее представление диаграммы предоставляет традиционное схематическое представление вашей модели. Оно показывает компоненты в виде блоков с соединениями между ними.

Современное представление диаграммы (UModernDiagramWidget)

Современное представление диаграммы предоставляет улучшенный визуальный редактор с улучшенным взаимодействием. Для подробной информации см. документацию Виджет Modern Diagram.

Использование

Добавление компонентов

  1. Просмотрите список классов компонентов слева
  2. Найдите компонент, который хотите добавить
  3. Дважды щелкните класс компонента, чтобы добавить его в модель
  4. Компонент появится в представлении диаграммы

Выбор компонентов

  • Одинарный щелчок: Выбрать компонент для просмотра его свойств
  • Двойной щелчок: Перейти внутрь компонента (если он содержит вложенные компоненты)

Навигация по модели

  • Используйте представление диаграммы для просмотра иерархической структуры вашей модели
  • Дважды щелкайте компоненты для перехода во вложенные структуры
  • Используйте кнопку "Назад" или контекстное меню для возврата вверх по иерархии

Обновление схемы

Диаграмма автоматически обновляется, когда:

  • Компоненты добавляются или удаляются
  • Создаются или удаляются соединения
  • Изменяются свойства компонентов (если они влияют на визуализацию)

Связанные виджеты

  • Виджет Modern Diagram - Улучшенная визуализация диаграммы
  • Виджет списка компонентов - Подробный браузер компонентов
  • Виджет связей компонентов - Управление соединениями компонентов
  • Виджет списка классов - Браузер классов компонентов
