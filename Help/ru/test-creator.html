Обзор

Виджет создания тестов позволяет создавать автоматизированные тесты для вашей модели нейронной сети. Вы можете определить тестовые случаи, указав свойства компонентов и их ожидаемые значения, которые затем можно выполнить для проверки поведения модели.

Доступ к виджету

Виджет создания тестов можно открыть из пункта меню Окно → Создатель тестов. Он появляется в отдельном окне.

Возможности

  • Выбор свойств: Выбор свойств компонентов для тестирования
  • Определение тестовых случаев: Определение тестовых случаев со значениями свойств
  • Браузер компонентов: Интегрированный список компонентов для удобного выбора свойств
  • Выполнение тестов: Запуск тестов для проверки поведения модели
  • Управление тестами: Добавление и удаление тестовых свойств

Использование

Создание теста

  1. Откройте виджет создания тестов
  2. Используйте список компонентов для выбора компонента
  3. Нажмите Добавить свойство для добавления свойства в тест
  4. Выберите свойство, которое хотите протестировать
  5. Установите ожидаемое значение или условие для свойства
  6. Повторите для дополнительных свойств
  7. Нажмите Создать тест для генерации теста

Добавление тестовых свойств

  • Выберите компонент из списка компонентов
  • Нажмите Добавить свойство
  • Перейдите к свойству, которое хотите протестировать
  • Свойство будет добавлено в список тестовых случаев

Удаление тестовых свойств

  1. Выберите свойство из списка тестовых случаев
  2. Нажмите Удалить выбранное свойство
  3. Свойство будет удалено из теста

Выполнение тестов

После создания теста вы можете выполнить его для проверки того, что ваша модель ведет себя как ожидается. Тест проверит, соответствуют ли указанные свойства их ожидаемым значениям.

Совет: Используйте тесты для проверки корректности модели после внесения изменений или для обеспечения согласованного поведения в различных конфигурациях модели.

Тестовые случаи

Тестовый случай состоит из:

  • Имени компонента
  • Имени свойства
  • Ожидаемого значения или условия

Несколько свойств можно протестировать в одном тестовом случае, что позволяет проверять сложное поведение модели.

Связанные виджеты

  • Виджет списка компонентов - Выбор компонентов для тестирования
  • Редактор свойств компонентов - Просмотр и редактирование значений свойств
