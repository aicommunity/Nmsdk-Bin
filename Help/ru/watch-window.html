Обзор

Окно наблюдения — это расширенный инструмент мониторинга, который позволяет создавать несколько графиков и диаграмм для наблюдения за поведением компонентов в реальном времени. Оно предоставляет гибкий интерфейс для мониторинга различных типов данных из вашей модели нейронной сети.

Доступ к виджету

Окно наблюдения можно открыть из пункта меню Окно → Окно наблюдения. Можно создать несколько окон наблюдения для мониторинга различных аспектов вашей модели.

Возможности

  • Несколько вкладок: Создание нескольких вкладок наблюдения, каждая со своим набором графиков
  • Сетка: Организация графиков в настраиваемой сетке (строки и столбцы)
  • Несколько типов графиков: Отображение различных типов визуализации данных
  • Серии данных: Добавление нескольких серий данных на каждый график
  • Обновления в реальном времени: Графики автоматически обновляются во время выполнения модели
  • Выбор свойств компонентов: Выбор любого свойства компонента для мониторинга
  • Управление интервалом обновления: Настройка частоты обновления графиков

Использование

Создание вкладки наблюдения

  1. Откройте окно наблюдения
  2. Используйте действие Создать вкладку из меню
  3. Будет создана новая вкладка
  4. Вы можете переименовать вкладки, дважды щелкнув на имени вкладки

Добавление графиков

  1. Выберите вкладку наблюдения
  2. Щелкните правой кнопкой мыши в области вкладки
  3. Выберите Добавить график или используйте контекстное меню
  4. Новый график будет добавлен в сетку

Настройка сетки графиков

Вы можете организовать графики в сетку:

  • Установите количество строк и столбцов
  • Графики будут расположены в указанной сетке
  • Изменяйте размер графиков, перетаскивая ручки разделителей

Добавление серий данных

  1. Щелкните на графике для его выбора
  2. Щелкните правой кнопкой мыши и выберите Добавить серию или используйте меню опций серий
  3. Перейдите к компоненту и свойству, которое хотите мониторить
  4. Настройте опции серии (цвет, имя и т.д.)
  5. Серию начнет отображать данные

Опции графика

Каждый график можно настроить с помощью:

  • Пользовательских меток и диапазонов осей
  • Нескольких серий данных с разными цветами
  • Интервалов обновления
  • Режимов отображения (непрерывный, скользящее окно и т.д.)

Опции серий

Каждую серию данных можно настроить с помощью:

  • Пользовательского имени и цвета
  • Источника данных (компонент и свойство)
  • Стиля отображения (линия, точки и т.д.)
  • Частоты обновления

Управление вкладками наблюдения

  • Создать вкладку: Добавить новую вкладку наблюдения
  • Закрыть вкладку: Закрыть вкладку, нажав кнопку X на вкладке
  • Переименовать вкладку: Дважды щелкните имя вкладки для переименования
  • Переместить вкладки: Перетащите вкладки для их переупорядочивания

Интервалы обновления

Вы можете управлять частотой обновления графиков:

  • Установите глобальный интервал обновления для всех графиков во вкладке
  • Настройте индивидуальные скорости обновления графиков
  • Балансируйте между обратной связью в реальном времени и производительностью
Совет: Используйте разные вкладки наблюдения для различных аспектов вашей модели. Например, одна вкладка для мониторинга входов, другая для анализа выходов и третья для наблюдения за внутренним состоянием.

Связанные виджеты

  • Виджет графика - Более простая визуализация одного графика
  • Виджет изображений - Мониторинг данных изображений
  • Виджет списка компонентов - Выбор компонентов для мониторинга
