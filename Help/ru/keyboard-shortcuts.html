Общие горячие клавиши

Клавиша Действие
F1 Открыть окно справки

Виджет Modern Diagram

Операции с узлами

Клавиша Действие
Delete Удалить выделенный(ые) компонент(ы) с подтверждением
Shift + Delete Удалить выделенный(ые) компонент(ы) без подтверждения
Esc Отменить текущую операцию (создание соединения и т.д.)

Операции с портами

Клавиша Действие
Enter Выбрать порт из окна выбора (когда открыт список портов)
Esc Закрыть окно выбора портов

Действия мышью

Виджет Modern Diagram

Действие Описание
Левый клик Выделить узел
Двойной левый клик Войти в компонент (переход к вложенным компонентам)
Правый клик Открыть контекстное меню
Левый клик + перетаскивание Выделить несколько узлов прямоугольником
Shift + Левый клик + перетаскивание Добавить узлы к выделению
Ctrl + Левый клик + перетаскивание Прокрутка холста
Колесо мыши Масштабирование
Двойной правый клик Подняться на уровень выше (выйти из вложенного компонента)
Левый клик на выходном порту Начать создание соединения
Перетаскивание на входной порт Завершить соединение
Shift + Левый клик на порту Показать порты на всю глубину вложенности (включая порты всех дочерних компонентов на всех уровнях)
