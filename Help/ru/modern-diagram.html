Виджет Modern Diagram

Виджет Modern Diagram предоставляет визуальный интерфейс для создания и управления соединениями между компонентами в вашей модели нейронной сети.

Обзор

Виджет Modern Diagram отображает компоненты в виде узлов с входными и выходными портами. Вы можете создавать соединения между компонентами, перетаскивая от выходных портов к входным.

Работа с узлами

Выделение узлов

  • Левый клик: Выделить один узел
  • Левый клик + перетаскивание: Выделить несколько узлов прямоугольником
  • Shift + Левый клик + перетаскивание: Добавить узлы к текущему выделению

Перемещение узлов

  • Перетаскивание: Переместить один узел
  • Перетаскивание выделенного: Переместить все выделенные узлы вместе как группу

Действия с узлами

  • Двойной левый клик: Войти внутрь компонента (переход к вложенным компонентам)
  • Правый клик: Открыть контекстное меню с дополнительными действиями
  • Клавиша Delete: Удалить выделенный(ые) компонент(ы) (с подтверждением)
  • Shift + Delete: Удалить без подтверждения

Работа с портами

Создание соединений

  • Левый клик на выходном порту: Начать создание соединения
  • Перетаскивание на входной порт: Завершить соединение
  • Правый клик: Отменить создание соединения
  • Клавиша Esc: Отменить создание соединения

Информация о портах

Наведите курсор на порт, чтобы увидеть подсказку с:

  • Именем порта и полным путем
  • Типом порта (Входной/Выходной)
  • Категорией порта (Own/Child/Alias)
  • Доступными действиями

Вложенные порты

  • Shift + Левый клик на порту: Показать вложенные порты (включая порты дочерних компонентов)
  • Клавиша Enter: Выбрать порт из окна выбора

Навигация по холсту

Прокрутка и масштабирование

  • Ctrl + Левый клик + перетаскивание: Прокрутка холста
  • Колесо мыши: Масштабирование (масштаб к позиции курсора)
  • Кнопка сброса масштаба: Сбросить масштаб до уровня по умолчанию

Выделение

  • Левый клик + перетаскивание на холсте: Выделить узлы прямоугольником
  • Shift + Левый клик + перетаскивание: Добавить к выделению

Навигация

  • Двойной правый клик: Подняться на уровень выше (выйти из вложенного компонента)

Подсказки (Tooltips)

Виджет Modern Diagram предоставляет подробные подсказки для всех интерактивных элементов:

  • Подсказки узлов: Показывают имя компонента, класс и доступные действия
  • Подсказки портов: Показывают детали порта и действия для соединений
  • Подсказки связей: Показывают информацию о соединении
  • Подсказки холста: Показывают общие действия на холсте

Подсказки автоматически адаптируются к текущему языку (английский или русский) в зависимости от настроек вашей системы.

Горячие клавиши

Клавиша Действие
Delete Удалить выделенный(ые) компонент(ы) с подтверждением
Shift + Delete Удалить без подтверждения
Esc Отменить текущую операцию (соединение и т.д.)
Enter Выбрать порт из окна выбора
