Обзор

Виджет изображений отображает данные изображений из компонентов вашей модели нейронной сети. Он особенно полезен для компонентов компьютерного зрения, позволяя визуализировать входы и выходы изображений в реальном времени во время выполнения модели.

Доступ к виджету

Виджет изображений можно открыть из пункта меню Окно → Изображения. Можно создать несколько окон изображений для одновременного мониторинга различных компонентов или свойств.

Возможности

  • Сетка: Отображение нескольких изображений в настраиваемой сетке (строки и столбцы)
  • Обновления в реальном времени: Изображения автоматически обновляются во время выполнения модели
  • Выбор изображения: Щелкните на изображении для выбора и детального просмотра
  • Полноэкранный просмотр: Просмотр выбранных изображений в полноэкранном режиме
  • Инструменты рисования: Рисование полигонов и прямоугольников на изображениях для аннотации
  • Сохранение изображений: Экспорт изображений в форматы BMP или JPEG
  • Индивидуальные каналы: Отображение изображений с использованием отдельных каналов вычислений
  • Режимы размера: Оригинальный размер, мозаика или пропорциональное масштабирование

Использование

Добавление изображений

  1. Откройте виджет изображений
  2. Щелкните правой кнопкой мыши в области виджета
  3. Выберите Добавить столбец или Добавить строку для создания слотов изображений
  4. Щелкните правой кнопкой мыши на слоте изображения и выберите Выбрать источник
  5. Выберите компонент и свойство, предоставляющее данные изображения

Настройка источников изображений

Каждый слот изображения можно настроить для отображения данных из определенного свойства компонента:

  • Выберите слот изображения
  • Щелкните правой кнопкой мыши и выберите Выбрать источник
  • Перейдите к компоненту и свойству, содержащему данные изображения
  • Изображение будет автоматически обновляться при изменении значения свойства

Макет изображений

  • Добавить столбец: Добавляет новый столбец в сетку изображений
  • Добавить строку: Добавляет новую строку в сетку изображений
  • Удалить столбец: Удаляет крайний правый столбец
  • Удалить строку: Удаляет нижнюю строку

Режимы размера изображений

  • Оригинальный размер: Отображение изображений в их исходном разрешении
  • Мозаика: Мозаичное размещение изображений для заполнения доступного пространства
  • Пропорционально: Пропорциональное масштабирование изображений для соответствия области отображения

Сохранение изображений

  • Сохранить в BMP: Сохранить выбранное изображение как файл BMP
  • Сохранить в JPEG: Сохранить выбранное изображение как файл JPEG
  • Сохранить все в BMP: Сохранить все изображения в сетке как файлы BMP
  • Сохранить все в JPEG: Сохранить все изображения в сетке как файлы JPEG

Рисование на изображениях

Виджет изображений поддерживает рисование полигонов и прямоугольников на изображениях:

  • Включите режим рисования для изображения
  • Рисуйте полигоны, щелкая точками на изображении
  • Рисуйте прямоугольники, используя левую и правую кнопки мыши
  • Выбирайте и изменяйте нарисованные фигуры

Индивидуальные каналы

Когда доступны несколько каналов вычислений, вы можете настроить каждое изображение для отображения данных из определенного канала. Используйте селектор каналов для переключения между каналами для каждого изображения.

Совет: Используйте несколько виджетов изображений для одновременного мониторинга различных аспектов вашей модели. Каждый виджет может отображать различный набор изображений.

Связанные виджеты

  • Окно наблюдения - Мониторинг других типов данных компонентов
  • Виджет графика - Визуализация числовых данных
  • Работа с компонентами - Общая информация о компонентах
