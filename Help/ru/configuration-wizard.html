Обзор

Мастер конфигурации проводит вас через процесс создания новой конфигурации проекта. Он предоставляет пошаговый интерфейс для настройки параметров проекта, выбора моделей, конфигурации каналов вычислений и указания настроек проекта.

Доступ к мастеру

Мастер конфигурации можно открыть из пункта меню Файл → Создать конфигурацию. Он появляется как многостраничный диалог мастера.

Возможности

  • Пошаговая настройка: Пошаговый процесс через несколько страниц конфигурации
  • Выбор типа проекта: Выбор между универсальным, анализом видео или моделированием
  • Выбор модели: Выбор из предопределенных моделей, загрузка из файла или создание из компонента
  • Конфигурация каналов: Настройка нескольких каналов вычислений с индивидуальными настройками
  • Выбор директории: Выбор директории проекта и расположений файлов
  • Конфигурация временного шага: Установка длительности временного шага по умолчанию

Шаги мастера

Шаг 1: Тип проекта

Выберите тип проекта, который хотите создать:

  • Универсальный: Проект общего назначения
  • Анализ видео: Проект, оптимизированный для задач анализа видео
  • Моделирование: Проект, ориентированный на моделирование нейронных сетей

Шаг 2: Выбор модели

Выберите, как получить модель для вашего проекта:

  • Предопределенная модель: Выбор из списка предопределенных моделей
  • Загрузить из файла: Загрузка модели из XML-файла
  • Модель из компонента: Создание модели на основе существующего компонента
  • Загрузить из коллекции моделей: Выбор из коллекции сохраненных моделей

Шаг 3: Директория проекта

Укажите, где сохранить ваш проект:

  • Просмотрите для выбора директории проекта
  • Мастер создаст необходимые поддиректории

Шаг 4: Конфигурация каналов

Настройте каналы вычислений:

  • Установите количество каналов вычислений
  • Настройте параметры для каждого канала индивидуально
  • Примените настройки ко всем каналам сразу (опционально)

Шаг 5: Временной шаг

Установите длительность временного шага по умолчанию для выполнения вашей модели.

Использование

Создание новой конфигурации

  1. Откройте мастера конфигурации из Файл → Создать конфигурацию
  2. Следуйте шагам мастера, заполняя необходимую информацию
  3. Используйте Далее для перехода к следующему шагу
  4. Используйте Назад для возврата к предыдущим шагам
  5. Нажмите Готово для создания конфигурации

Настройки каналов

При настройке нескольких каналов:

  • Выберите канал из списка для его настройки
  • Каждый канал может иметь разные настройки
  • Используйте "Применить ко всем каналам" для использования одинаковых настроек для всех каналов
Совет: Вы всегда можете изменить настройки проекта после создания, используя диалог опций проекта.

Связанные виджеты

  • Виджет каналов вычислений - Управление каналами после создания проекта
  • Начало работы - Руководство по базовой настройке проекта
