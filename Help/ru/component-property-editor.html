Обзор

Редактор свойств компонентов позволяет просматривать и изменять свойства выбранных компонентов в вашей модели нейронной сети. Он предоставляет интерфейс для изменения параметров компонентов, просмотра значений свойств и управления конфигурациями компонентов.

Доступ к виджету

Редактор свойств компонентов обычно отображается в док-виджете или панели в главном окне приложения. Он автоматически обновляется при выборе компонента из списка компонентов или представления диаграммы.

Возможности

  • Отображение свойств: Показывает все свойства выбранного компонента
  • Редактирование свойств: Позволяет изменять значения свойств напрямую
  • Выбор компонентов: Интегрирован с виджетом списка компонентов для удобного выбора
  • XML-представление: Просмотр и редактирование свойств компонентов в формате XML
  • Настройки области видимости: Установка свойств для текущего компонента, глобально или для владельца компонента

Использование

Выбор компонента

  1. Выберите компонент из виджета Список компонентов
  2. Редактор свойств автоматически отобразит все свойства выбранного компонента
  3. Свойства организованы по категориям (Параметры, Состояние, Входы, Выходы)

Редактирование свойств

  1. Перейдите к свойству, которое хотите изменить
  2. Щелкните на значении свойства для редактирования
  3. Введите новое значение
  4. Используйте кнопку Установить для применения изменений к текущему компоненту

Установка области видимости свойства

  • Установить: Применяет значение свойства к текущему выбранному компоненту
  • Установить глобально: Устанавливает значение свойства глобально для всех компонентов этого типа
  • Установить глобально для владельца: Устанавливает значение свойства для владельца компонента и всех его дочерних элементов

Просмотр XML

Нажмите кнопку Показать XML для просмотра свойств компонента в формате XML. Это полезно для опытных пользователей, которым нужно увидеть исходную структуру свойств.

Категории свойств

  • Параметры: Настраиваемые параметры, влияющие на поведение компонента
  • Состояние: Внутренние переменные состояния компонента
  • Входы: Входные свойства, получающие данные от других компонентов
  • Выходы: Выходные свойства, отправляющие данные другим компонентам
Совет: Изменения свойств вступают в силу немедленно. Вы можете видеть результаты в реальном времени, если ваша модель запущена.
