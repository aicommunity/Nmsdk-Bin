Обзор

Виджет списка компонентов предоставляет иерархическое древовидное представление всех компонентов в вашей модели нейронной сети. Он отображает компоненты в порядке вычисления, показывая вложенную структуру вашей модели, и позволяет просматривать и выбирать компоненты для просмотра их свойств.

Доступ к виджету

Виджет списка компонентов обычно отображается в док-панели главного окна приложения. Он автоматически создается при запуске приложения и является одним из основных интерфейсов для навигации по компонентам.

Возможности

  • Иерархическое дерево: Отображает компоненты в древовидной структуре, показывающей отношения родитель-потомок
  • Порядок вычисления: Компоненты перечислены в порядке их вычисления
  • Выбор компонентов: Щелчок для выбора компонентов и просмотра их свойств
  • Вкладки свойств: Просмотр параметров, состояния, входов, выходов и избранного
  • Редактирование свойств: Редактирование свойств компонентов напрямую из списка
  • Фильтрация компонентов: Поиск и фильтрация компонентов по имени
  • Выбор канала: Переключение между каналами вычислений
  • Контекстное меню: Доступ к операциям с компонентами (удаление, переименование, сброс и т.д.)

Использование

Просмотр компонентов

  • Разверните узлы дерева, чтобы увидеть вложенные компоненты
  • Сверните узлы, чтобы скрыть подкомпоненты
  • Используйте поле фильтра для поиска конкретных компонентов
  • Компоненты отображаются в порядке вычисления

Выбор компонентов

  1. Щелкните на компоненте в дереве для его выбора
  2. Свойства компонента будут отображены во вкладках свойств
  3. Выбранный компонент также будет выделен в представлении диаграммы

Просмотр свойств компонентов

Виджет предоставляет несколько вкладок для просмотра информации о компонентах:

  • Параметры: Настраиваемые параметры компонента
  • Состояние: Внутренние переменные состояния
  • Входы: Входные свойства, получающие данные
  • Выходы: Выходные свойства, отправляющие данные
  • Избранное: Отмеченные избранные свойства для быстрого доступа

Редактирование свойств

  1. Выберите компонент
  2. Перейдите на соответствующую вкладку свойств
  3. Щелкните на значении свойства для редактирования
  4. Введите новое значение и нажмите Enter
  5. Изменения вступают в силу немедленно

Операции с компонентами

Щелкните правой кнопкой мыши на компоненте для доступа к контекстному меню с операциями:

  • Переместить вверх/вниз: Изменить порядок вычисления компонента (Shift+Вверх/Вниз)
  • Переименовать: Переименовать компонент
  • Удалить: Удалить компонент из модели
  • Копировать имя: Копировать имя компонента в буфер обмена
  • Сбросить: Сбросить компонент в начальное состояние
  • Вычислить: Принудительно выполнить шаг вычисления
  • Инициализировать/Деинициализировать: Инициализировать или деинициализировать компонент

Фильтрация компонентов

Используйте поле фильтра вверху для поиска компонентов:

  • Введите часть имени компонента
  • Дерево отфильтруется, чтобы показать только соответствующие компоненты
  • Очистите фильтр, чтобы снова показать все компоненты

Выбор канала

Когда существует несколько каналов вычислений:

  • Используйте селектор каналов для переключения между каналами
  • Список компонентов обновляется для отображения компонентов выбранного канала
  • Значения свойств отражают состояние выбранного канала

Горячие клавиши

  • Shift + Стрелка вверх: Переместить выбранный компонент вверх в порядке вычисления
  • Shift + Стрелка вниз: Переместить выбранный компонент вниз в порядке вычисления
Совет: Виджет списка компонентов синхронизирован с представлением диаграммы. Выбор компонента в одном представлении выделяет его в другом.
