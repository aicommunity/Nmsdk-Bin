Обзор

Виджет TCP-сервера позволяет управлять и мониторить TCP-сервер, который может взаимодействовать с вашей моделью нейронной сети. Это позволяет внешним приложениям взаимодействовать с вашей моделью через сетевое соединение.

Доступ к виджету

Виджет TCP-сервера можно открыть из пункта меню Окно → TCP-сервер. Он появляется в отдельном окне или док-панели.

Возможности

  • Управление сервером: Запуск и остановка TCP-сервера
  • Конфигурация: Настройка параметров сервера (порт, адрес и т.д.)
  • Мониторинг статуса: Просмотр статуса сервера и информации о соединениях
  • Автоперезапуск: Автоматический перезапуск сервера при ошибках
  • Сетевое взаимодействие: Включение взаимодействия внешних приложений с вашей моделью

Использование

Запуск сервера

  1. Откройте виджет TCP-сервера
  2. Настройте параметры сервера (при необходимости)
  3. Нажмите Запустить сервер
  4. Сервер начнет прослушивать соединения

Остановка сервера

  • Нажмите Остановить сервер для остановки сервера
  • Все активные соединения будут закрыты

Настройка параметров сервера

Настройте сервер перед запуском:

  • Установите номер порта сервера
  • Настройте сетевой адрес (при необходимости)
  • Установите другие параметры сервера
  • Нажмите Применить для сохранения настроек

Сброс сервера

Используйте кнопку Сбросить для сброса состояния сервера и очистки любых ошибок.

Статус сервера

Виджет отображает:

  • Текущее состояние сервера (запущен/остановлен)
  • Количество активных соединений
  • Порт и адрес сервера
  • Сообщения об ошибках (если есть)

Функция автоперезапуска

Сервер можно настроить для автоматического перезапуска при возникновении ошибок или неожиданной остановке. Это помогает поддерживать связь для внешних приложений.

Предупреждение: Убедитесь, что порт сервера не используется другим приложением. Использование уже занятого порта предотвратит запуск сервера.

Сетевое взаимодействие

После запуска сервера внешние приложения могут:

  • Подключаться к серверу, используя настроенный порт
  • Отправлять команды для управления моделью
  • Получать данные из модели
  • Мониторить состояние модели в реальном времени

Связанные виджеты

  • Панель статуса - Общий статус системы
  • Виджет журнала - Просмотр сообщений журнала сервера
