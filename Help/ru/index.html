Руководство пользователя NMSDK

Добро пожаловать в руководство пользователя NMSDK (Neuro Modeler SDK). Это руководство поможет вам начать работу с приложением NMSDK и его функциями.

Содержание

  • Начало работы
  • Работа с компонентами
  • Виджет Modern Diagram
  • Горячие клавиши

О NMSDK

NMSDK — это комплект средств разработки для моделирования нейронных сетей и робототехнических систем. Он предоставляет компонентную архитектуру для создания, конфигурирования и выполнения моделей нейронных сетей, включая импульсные нейронные сети, системы управления движением и компоненты компьютерного зрения.

Быстрый старт

Чтобы быстро начать работу:

  1. Создайте новый проект или откройте существующий
  2. Добавьте компоненты в вашу модель с помощью окна управления компонентами
  3. Соедините компоненты с помощью виджета Modern Diagram
  4. Настройте свойства компонентов
  5. Запустите вашу модель
Совет: Нажмите F1 в любое время, чтобы открыть это окно справки, или используйте пункт меню Справка → Руководство пользователя.

Основные возможности

  • Система компонентов: Создание и управление компонентами нейронных сетей
  • Modern Diagram: Визуальный редактор для соединения компонентов
  • Мониторинг в реальном времени: Окна наблюдения за поведением компонентов
  • Управление конфигурациями: Сохранение и загрузка конфигураций проектов
