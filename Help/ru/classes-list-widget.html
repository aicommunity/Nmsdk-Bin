Обзор

Виджет списка классов отображает иерархический список доступных классов компонентов, которые можно добавить в вашу модель нейронной сети. Он интегрирован в виджет движка рисования и предоставляет легкий доступ ко всем типам компонентов в системе.

Доступ к виджету

Виджет списка классов отображается в левой панели виджета движка рисования. Он всегда виден при работе с представлением диаграммы.

Возможности

  • Иерархическая организация: Классы компонентов организованы по категориям
  • Функция поиска: Поиск конкретных классов компонентов
  • Быстрое добавление: Двойной щелчок для добавления компонентов в модель
  • Просмотр категорий: Навигация по категориям компонентов
  • Интеграция: Работает бесшовно с представлением диаграммы

Использование

Просмотр классов компонентов

  • Разверните узлы категорий, чтобы увидеть классы компонентов в них
  • Навигация по иерархии для поиска нужного компонента
  • Используйте поиск для быстрого поиска конкретных компонентов

Добавление компонентов

  1. Найдите класс компонента, который хотите добавить
  2. Дважды щелкните на имени класса компонента
  3. Компонент будет добавлен в вашу модель
  4. Новый компонент появится в представлении диаграммы

Поиск компонентов

Используйте поле поиска для поиска компонентов:

  • Введите часть имени класса компонента
  • Список отфильтруется для показа соответствующих компонентов
  • Очистите поиск, чтобы снова показать все компоненты

Категории компонентов

Компоненты организованы в категории, такие как:

  • Входные компоненты
  • Обрабатывающие компоненты
  • Выходные компоненты
  • Слои нейронных сетей
  • Компоненты компьютерного зрения
  • Компоненты управления движением
Совет: Список классов синхронизирован с представлением диаграммы. Когда вы добавляете компонент, он сразу появляется на диаграмме.
