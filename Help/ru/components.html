Обзор компонентов

Компоненты — это строительные блоки моделей нейронных сетей в NMSDK. Каждый компонент представляет определенную функцию или блок обработки в вашей модели.

Типы компонентов

  • Входные компоненты: Принимают внешние данные или сигналы
  • Обрабатывающие компоненты: Выполняют вычисления или преобразования
  • Выходные компоненты: Генерируют результаты или управляющие сигналы
  • Компоненты нейронных сетей: Реализуют слои и операции нейронных сетей

Свойства компонентов

Каждый компонент имеет настраиваемые свойства, которые контролируют его поведение:

  • Параметры: Значения конфигурации, влияющие на работу компонента
  • Входы: Данные или сигналы, принимаемые компонентом
  • Выходы: Данные или сигналы, генерируемые компонентом
  • Состояние: Внутренние переменные состояния компонента

Управление компонентами

Добавление компонентов

Компоненты можно добавлять в модель через окно управления компонентами или перетаскиванием из списка классов.

Настройка компонентов

Выберите компонент и используйте редактор свойств для изменения его параметров. Изменения вступают в силу немедленно.

Соединение компонентов

Используйте виджет Modern Diagram для создания соединений между портами компонентов. См. раздел Виджет Modern Diagram для подробностей.

Удаление компонентов

Выберите компонент и нажмите Delete (с подтверждением) или Shift + Delete (без подтверждения).

Предупреждение: Удаление компонента также удалит все соединения к этому компоненту и от него.

Иерархия компонентов

Компоненты могут содержать вложенные подкомпоненты, создавая иерархическую структуру. Используйте двойной клик для навигации во вложенные компоненты в представлении Modern Diagram.
