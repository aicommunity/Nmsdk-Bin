Overview

The Configuration Wizard guides you through the process of creating a new project configuration. It provides a step-by-step interface for setting up project parameters, selecting models, configuring calculation channels, and specifying project settings.

Accessing the Wizard

The Configuration Wizard can be opened from the File → Create Config menu item. It appears as a multi-page wizard dialog.

Features

  • Step-by-step Setup: Guided process through multiple configuration pages
  • Project Type Selection: Choose between Universal, Video Analysis, or Modeling project types
  • Model Selection: Select from predefined models, load from file, or create from component
  • Channel Configuration: Set up multiple calculation channels with individual settings
  • Directory Selection: Choose project directory and file locations
  • Time Step Configuration: Set default time step duration

Wizard Steps

Step 1: Project Type

Select the type of project you want to create:

  • Universal: General-purpose project
  • Video Analysis: Project optimized for video analysis tasks
  • Modeling: Project focused on neural network modeling

Step 2: Model Selection

Choose how to obtain the model for your project:

  • Predefined Model: Select from a list of predefined models
  • Load from File: Load a model from an XML file
  • Model from Component: Create a model based on an existing component
  • Load from Models Collection: Select from a collection of saved models

Step 3: Project Directory

Specify where to save your project:

  • Browse to select a project directory
  • The wizard will create necessary subdirectories

Step 4: Channel Configuration

Configure calculation channels:

  • Set the number of calculation channels
  • Configure settings for each channel individually
  • Apply settings to all channels at once (optional)

Step 5: Time Step

Set the default time step duration for your model execution.

Usage

Creating a New Configuration

  1. Open the Configuration Wizard from File → Create Config
  2. Follow the wizard steps, filling in required information
  3. Use Next to proceed to the next step
  4. Use Back to return to previous steps
  5. Click Finish to create the configuration

Channel Settings

When configuring multiple channels:

  • Select a channel from the list to configure it
  • Each channel can have different settings
  • Use "Apply to All Channels" to use the same settings for all channels
Tip: You can always modify project settings after creation using the project options dialog.
