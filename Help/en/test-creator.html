Overview

The Test Creator Widget allows you to create automated tests for your neural network model. You can define test cases by specifying component properties and their expected values, which can then be executed to verify model behavior.

Accessing the Widget

The Test Creator Widget can be opened from the Window → Test Creator menu item. It appears in a separate window.

Features

  • Property Selection: Select component properties to test
  • Test Case Definition: Define test cases with property values
  • Component Browser: Integrated component list for easy property selection
  • Test Execution: Run tests to verify model behavior
  • Test Management: Add and remove test properties

Usage

Creating a Test

  1. Open the Test Creator Widget
  2. Use the component list to select a component
  3. Click Add Property to add a property to the test
  4. Select the property you want to test
  5. Set the expected value or condition for the property
  6. Repeat for additional properties
  7. Click Create Test to generate the test

Adding Test Properties

  • Select a component from the component list
  • Click Add Property
  • Navigate to the property you want to test
  • The property will be added to the test case list

Removing Test Properties

  1. Select a property from the test case list
  2. Click Delete Selected Property
  3. The property will be removed from the test

Executing Tests

After creating a test, you can execute it to verify that your model behaves as expected. The test will check if the specified properties match their expected values.

Tip: Use tests to verify model correctness after making changes, or to ensure consistent behavior across different model configurations.

Test Cases

A test case consists of:

  • Component name
  • Property name
  • Expected value or condition

Multiple properties can be tested in a single test case, allowing you to verify complex model behavior.

Related Widgets

  • Components List Widget - Select components for testing
  • Component Property Editor - View and edit property values
