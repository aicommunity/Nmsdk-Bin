About NMSDK

NMSDK is a Software Development Kit for modeling neural networks and robotic systems. It provides a component-based architecture for creating, configuring, and executing neural network models, including spiking neural networks, motion control systems, and computer vision components.

Quick Start

To get started quickly:

  1. Create a new project or open an existing one
  2. Add components to your model using the Components Control window
  3. Connect components using the Modern Diagram Widget
  4. Configure component properties
  5. Run your model

Main Features

  • Component System: Create and manage neural network components
  • Modern Diagram: Visual editor for component connections
  • Real-time Monitoring: Watch windows for observing component behavior
  • Configuration Management: Save and load project configurations
