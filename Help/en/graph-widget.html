Overview

The Graph Widget provides real-time visualization of numerical data from your neural network model. It displays time-series graphs showing how component properties change over time during model execution.

Accessing the Widget

The Graph Widget can be opened from the Window menu. It appears in a separate window or dock panel, allowing you to monitor multiple data series simultaneously.

Features

  • Multiple Series: Display multiple data series on the same graph
  • Real-time Updates: Graphs update automatically as data arrives
  • Customizable Axes: Set custom labels and ranges for X and Y axes
    • Color Coding: Each series can have a different color
  • Data Source Selection: Choose component properties to monitor
  • Auto-scaling: Automatic or manual axis scaling
  • Matrix Support: Display matrix data with X and Y indices

Usage

Adding a Data Series

  1. Click the Select Output button
  2. Navigate to the component and property you want to monitor
  3. For matrix properties, specify the X and Y indices
  4. Select the calculation channel (if multiple channels exist)
  5. The graph will start displaying data from the selected property

Configuring Graph Settings

  • Graph Settings: Click the settings button to configure axis labels, ranges, and display options
  • Color Selection: Change the color of individual graph series
  • Series Name: Each series can have a custom name for identification

Graph Layout

The Graph Widget supports:

  • Custom X and Y axis labels
  • Manual or automatic axis limits
  • Display of last N elements (sliding window)
  • Auto-updating right border (for time-series)

Deleting Series

  • Select a series from the list
  • Click Delete Current Item to remove it
  • Use Delete All to clear all series

Data Sources

The Graph Widget can monitor:

  • Scalar properties (single values)
  • Matrix properties (with X and Y indices)
  • Properties from any calculation channel
  • Multiple properties simultaneously

Graph Display Modes

  • Continuous Update: Graph updates in real-time as new data arrives
  • Sliding Window: Display only the last N data points
  • Fixed Range: Display data within a fixed time or value range
Tip: Use multiple Graph Widgets to monitor different aspects of your model. Each widget can display different properties or use different display settings.

Related Widgets

  • Watch Window - Advanced monitoring with multiple chart types
  • Images Widget - Visualize image data
  • Profiling Widget - Performance analysis
