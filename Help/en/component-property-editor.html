Overview

The Component Property Editor allows you to view and modify properties of selected components in your neural network model. It provides an interface for changing component parameters, viewing property values, and managing component configurations.

Accessing the Widget

The Component Property Editor is typically displayed in a dock widget or panel within the main application window. It automatically updates when you select a component from the Components List or the diagram view.

Features

  • Property Display: Shows all properties of the selected component
  • Property Editing: Allows you to modify property values directly
  • Component Selection: Integrated with the Components List widget for easy component selection
  • XML View: View and edit component properties in XML format
  • Scope Settings: Set properties for current component, globally, or for component owner

Usage

Selecting a Component

  1. Select a component from the Components List widget
  2. The Property Editor will automatically display all properties of the selected component
  3. Properties are organized by category (Parameters, State, Inputs, Outputs)

Editing Properties

  1. Navigate to the property you want to modify
  2. Click on the property value to edit it
  3. Enter the new value
  4. Use the Set button to apply changes to the current component

Setting Property Scope

  • Set: Applies the property value to the currently selected component
  • Set Global: Sets the property value globally for all components of this type
  • Set Global Owner: Sets the property value for the component owner and all its children

Viewing XML

Click the Show XML button to view the component's properties in XML format. This is useful for advanced users who need to see the raw property structure.

Property Categories

  • Parameters: Configurable parameters that affect component behavior
  • State: Internal state variables of the component
  • Inputs: Input properties that receive data from other components
  • Outputs: Output properties that send data to other components
Tip: Property changes take effect immediately. You can see the results in real-time if your model is running.

Related Widgets

  • Components List Widget - Select components to edit

  • Modern Diagram Widget - Visual component editor
