Component Links Widget

Overview

The Component Links Widget provides a detailed interface for managing connections between components in your neural network model. It displays inputs, outputs, and existing links for components, allowing you to create, modify, and delete connections with precision.

Accessing the Widget

The Component Links Widget can be opened from the main window menu or by right-clicking on a component in the diagram view. It appears in a separate window or dock panel.

Features

  • Three Operation Modes: Single component view, two-component linking, and link transfer mode
  • Hierarchical Display: Shows nested inputs/outputs including child component ports
  • Link Management: Create, break, and switch connections between components
  • Public Ports Only: Displays only publicly accessible inputs and outputs
  • Internal Links Filter: Option to hide or show internal component links

Operation Modes

Mode 1: Single Component View

In this mode, the widget displays all inputs and outputs (including nested ones) of a single component. You can:

  • View all connections for the component
  • Create links between the component's own inputs and outputs
  • Create links between nested components within the selected component

Mode 2: Two-Component Linking

This mode displays outputs and links (including nested) of the first component, and inputs (including nested) of the second component. Use this mode to:

  • Create new connections between two specific components
  • View existing connections between the components
  • Modify connections between the two components

Mode 3: Link Transfer Mode

This special mode shows only available outputs from both components (including nested outputs and links). Use this mode to:

  • Transfer connections from one component to another
  • Copy link patterns between similar components

Usage

Viewing Component Links

  1. Right-click on a component in the diagram view
  2. Select "View Links" or "Component Links" from the context menu
  3. The Component Links Widget opens showing the component's inputs, outputs, and connections

Creating a Link

  1. Open the Component Links Widget for the components you want to connect
  2. Select an output from the source component
  3. Select an input from the destination component
  4. Click the Create Link button

Breaking a Link

  1. Select the link you want to remove from the links table
  2. Click the Break Link button
  3. The connection will be removed

Switching a Link

Use the Switch Link button to redirect an existing connection to a different input or output.

Hiding Internal Links

Use the "Hide Internal Links" checkbox to filter out internal component connections and focus on external links only.

Note: The widget only displays public inputs and outputs. Private or internal ports are not shown.
