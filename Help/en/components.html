Component Overview

Components are the building blocks of neural network models in NMSDK. Each component represents a specific function or processing unit in your model.

Component Types

  • Input Components: Receive external data or signals
  • Processing Components: Perform computations or transformations
  • Output Components: Generate results or control signals
  • Neural Network Components: Implement neural network layers and operations

Component Properties

Each component has configurable properties that control its behavior:

  • Parameters: Configuration values that affect component operation
  • Inputs: Data or signals received by the component
  • Outputs: Data or signals produced by the component
  • State: Internal state variables of the component

Managing Components

Adding Components

Components can be added to your model through the Components Control window or by dragging from the class list.

Configuring Components

Select a component and use the property editor to modify its parameters. Changes take effect immediately.

Connecting Components

Use the Modern Diagram Widget to create connections between component ports. See the Modern Diagram Widget section for details.

Deleting Components

Select a component and press Delete (with confirmation) or Shift + Delete (without confirmation).

Warning: Deleting a component will also remove all connections to and from that component.

Component Hierarchy

Components can contain nested sub-components, creating a hierarchical structure. Use double-click to navigate into nested components in the Modern Diagram view.
