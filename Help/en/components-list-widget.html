Components List Widget

  • ← Back to Main Page

Overview

The Components List Widget provides a hierarchical tree view of all components in your neural network model. It displays components in calculation order, showing the nested structure of your model, and allows you to browse and select components to view their properties.

Accessing the Widget

The Components List Widget is typically displayed in a dock panel in the main application window. It's automatically created when the application starts and is one of the primary interfaces for component navigation.

Features

  • Hierarchical Tree: Displays components in a tree structure showing parent-child relationships
  • Calculation Order: Components are listed in the order they are calculated
  • Component Selection: Click to select components and view their properties
  • Property Tabs: View parameters, state, inputs, outputs, and favorites
  • Property Editing: Edit component properties directly from the list
  • Component Filtering: Search and filter components by name
  • Channel Selection: Switch between calculation channels
  • Context Menu: Access component operations (delete, rename, reset, etc.)

Usage

Browsing Components

  • Expand tree nodes to see nested components
  • Collapse nodes to hide sub-components
  • Use the filter box to search for specific components
  • Components are displayed in calculation order

Selecting Components

  1. Click on a component in the tree to select it
  2. The component's properties will be displayed in the property tabs
  3. The selected component will also be highlighted in the diagram view

Viewing Component Properties

The widget provides several tabs for viewing component information:

  • Parameters: Configurable parameters of the component
  • State: Internal state variables
  • Inputs: Input properties that receive data
  • Outputs: Output properties that send data
  • Favorites: Marked favorite properties for quick access

Editing Properties

  1. Select a component
  2. Navigate to the appropriate property tab
  3. Click on a property value to edit it
  4. Enter the new value and press Enter
  5. Changes take effect immediately

Component Operations

Right-click on a component to access the context menu with operations:

  • Move Up/Down: Change component calculation order (Shift+Up/Down)
  • Rename: Rename the component
  • Delete: Remove the component from the model
  • Copy Name: Copy component name to clipboard
  • Reset: Reset component to initial state
  • Calculate: Force a calculation step
  • Init/UnInit: Initialize or uninitialize the component

Filtering Components

Use the filter box at the top to search for components:

  • Type part of a component name
  • The tree will filter to show only matching components
  • Clear the filter to show all components again

Channel Selection

When multiple calculation channels exist:

  • Use the channel selector to switch between channels
  • The component list updates to show components for the selected channel
  • Property values reflect the state of the selected channel

Keyboard Shortcuts

  • Shift + Up Arrow: Move selected component up in calculation order
  • Shift + Down Arrow: Move selected component down in calculation order
Tip: The Components List Widget is synchronized with the diagram view. Selecting a component in one view highlights it in the other.
