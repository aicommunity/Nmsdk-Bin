Calculation Channels Widget

  • ← Back to Main Page

Overview

The Calculation Channels Widget allows you to manage multiple calculation channels in your neural network model. Each channel can run independently, allowing you to test different configurations or run parallel simulations.

Accessing the Widget

The Calculation Channels Widget is displayed in a dock panel in the main application window. You can show or hide it using the Window → Channels Control menu item.

Features

  • Multiple Channels: Create and manage multiple independent calculation channels
  • Channel Control: Start, pause, reset, and delete channels individually
  • Channel Management: Add, insert, delete, and clone channels
  • Independent Execution: Each channel runs its own instance of the model
  • Channel Selection: Switch between channels to view and modify their state

Usage

Adding a Channel

  1. Click the Add Channel button or use the context menu
  2. A new channel will be created and added to the list
  3. The new channel starts in a stopped state

Inserting a Channel

Use Insert Channel to add a new channel at a specific position in the list.

Cloning a Channel

  1. Select the channel you want to clone
  2. Click Clone Channel or use the context menu
  3. A copy of the channel with all its settings will be created

Deleting a Channel

  1. Select the channel you want to delete
  2. Click Delete Selected Channel or use the context menu
  3. Confirm the deletion

Controlling Channel Execution

  • Start Channel: Begins execution of the selected channel
  • Pause Channel: Temporarily stops execution (can be resumed)
  • Reset Channel: Resets the channel to its initial state

Selecting a Channel

Click on a channel in the list to select it. The selected channel becomes the active channel, and other widgets will display information for this channel.

Tip: You can run multiple channels simultaneously to compare different model configurations or test parallel processing scenarios.

Channel Operations

All channel operations are available through:

  • Buttons in the widget interface
  • Context menu (right-click on a channel)
  • Menu items in the Channels menu

Related Widgets

  • Status Panel - Displays current channel status
  • Components List Widget - Shows components for the selected channel
  • Watch Window - Monitor channel data in real-time
