Images Widget

Overview

The Images Widget displays image data from components in your neural network model. It's particularly useful for computer vision components, allowing you to visualize image inputs and outputs in real-time during model execution.

Accessing the Widget

The Images Widget can be opened from the Window → Images menu item. Multiple image windows can be created to monitor different components or properties simultaneously.

Features

  • Grid Layout: Display multiple images in a customizable grid (rows and columns)
  • Real-time Updates: Images update automatically during model execution
  • Image Selection: Click on an image to select and view it in detail
  • Full-screen View: View selected images in full-screen mode
  • Drawing Tools: Draw polygons and rectangles on images for annotation
  • Save Images: Export images to BMP or JPEG format
  • Individual Channels: Display images using individual calculation channels
  • Size Modes: Original size, tiled, or proportional scaling

Usage

Adding Images

  1. Open the Images Widget
  2. Right-click in the widget area
  3. Select Add Column or Add Row to create image slots
  4. Right-click on an image slot and select Select Source
  5. Choose the component and property that provides the image data

Configuring Image Sources

Each image slot can be configured to display data from a specific component property:

  • Select the image slot
  • Right-click and choose Select Source
  • Navigate to the component and property containing image data
  • The image will automatically update when the property value changes

Image Layout

  • Add Column: Adds a new column to the image grid
  • Add Row: Adds a new row to the image grid
  • Delete Column: Removes the rightmost column
  • Delete Row: Removes the bottom row

Image Size Modes

  • Original Size: Display images at their native resolution
  • Tiled: Tile images to fill available space
  • Proportional: Scale images proportionally to fit the display area

Saving Images

  • Save to BMP: Save the selected image as a BMP file
  • Save to JPEG: Save the selected image as a JPEG file
  • Save All to BMP: Save all images in the grid as BMP files
  • Save All to JPEG: Save all images in the grid as JPEG files

Drawing on Images

The Images Widget supports drawing polygons and rectangles on images:

  • Enable drawing mode for an image
  • Draw polygons by clicking points on the image
  • Draw rectangles using left and right mouse buttons
  • Select and modify drawn shapes

Individual Channels

When multiple calculation channels are available, you can configure each image to display data from a specific channel. Use the channel selector to switch between channels for each image.

Tip: Use multiple Images Widgets to monitor different aspects of your model simultaneously. Each widget can display a different set of images.

Related Widgets
