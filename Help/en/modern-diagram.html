Modern Diagram Widget

The Modern Diagram Widget provides a visual interface for creating and managing connections between components in your neural network model.

  • ← Back to Main Page

Overview

The Modern Diagram Widget displays components as nodes with input and output ports. You can create connections between components by dragging from output ports to input ports.

Working with Nodes

Selecting Nodes

  • Left Click: Select a single node
  • Left Click + Drag: Select multiple nodes with a rectangle
  • Shift + Left Click + Drag: Add nodes to the current selection

Moving Nodes

  • Drag: Move a single node
  • Drag Selected: Move all selected nodes together as a group

Node Actions

  • Double Left Click: Enter into the component (navigate to nested components)
  • Right Click: Open context menu with additional actions
  • Delete Key: Delete selected component(s) (with confirmation)
  • Shift + Delete: Delete without confirmation

Working with Ports

Creating Connections

  • Left Click on Output Port: Start creating a connection
  • Drag to Input Port: Complete the connection
  • Right Click: Cancel connection creation
  • Esc Key: Cancel connection creation

Port Information

Hover over a port to see a tooltip with:

  • Port name and full path
  • Port type (Input/Output)
  • Port category (Own/Child/Alias)
  • Available actions

Nested Ports

  • Shift + Left Click on Port: Show nested ports (including child component ports)
  • Enter Key: Select port from the selection window

Canvas Navigation

Panning and Zooming

  • Ctrl + Left Click + Drag: Pan the canvas
  • Mouse Wheel: Zoom in/out (zoom to cursor position)
  • Reset Zoom Button: Reset zoom to default level

Selection

  • Left Click + Drag on Canvas: Select nodes with rectangle
  • Shift + Left Click + Drag: Add to selection

Navigation

  • Double Right Click: Go up one level (exit nested component)

Tooltips

The Modern Diagram Widget provides comprehensive tooltips for all interactive elements:

  • Node Tooltips: Show component name, class, and available actions
  • Port Tooltips: Show port details and connection actions
  • Link Tooltips: Show connection information
  • Canvas Tooltips: Show general canvas actions

Tooltips automatically adapt to the current language (English or Russian) based on your system settings.

Keyboard Shortcuts

Key Action
Delete Delete selected component(s) with confirmation
Shift + Delete Delete without confirmation
Esc Cancel current operation (connection, etc.)
Enter Select port from selection window
