Overview

The Class Editor allows you to view and edit properties of component classes in the system. It provides an interface for examining class definitions, their properties, methods, and documentation.

Accessing the Widget

The Class Editor can be opened from the Window → Classes Description menu item. It appears in a separate window.

Features

  • Class Selection: Browse and select component classes from a list
  • Class Information Display: View detailed information about selected classes
  • Property Viewing: See all properties, methods, and attributes of a class
  • Documentation: View class documentation and descriptions
  • Class Hierarchy: Understand class inheritance and relationships

Usage

Viewing Class Information

  1. Open the Class Editor
  2. Browse the class list on the left side
  3. Select a class to view its information
  4. Class details will be displayed on the right side

Class List

The left panel shows a hierarchical list of all available component classes:

  • Navigate through categories to find classes
  • Click on a class to select it
  • The selected class information appears in the display panel

Class Information Display

The right panel displays detailed information about the selected class:

  • Class name and description
  • Inheritance hierarchy
  • Properties and their types
  • Methods and functions
  • Documentation and usage notes

Class Properties

For each class, you can view:

  • Parameters: Configurable parameters of the class
  • Inputs: Input properties that the class accepts
  • Outputs: Output properties that the class produces
  • State Variables: Internal state variables
  • Methods: Available methods and functions
Tip: Use the Class Editor to understand component capabilities before adding them to your model, or to learn about available properties and methods.
