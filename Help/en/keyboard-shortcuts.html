Node Operations

Shortcut Action

Port Operations

Shortcut Action
Enter Select port from selection window (when port list is open)

Mouse Actions

Modern Diagram Widget

Action Description
Left Click Select node
Double Left Click Enter component (navigate to nested components)
Right Click Open context menu
Left Click + Drag Select multiple nodes with rectangle
Shift + Left Click + Drag Add nodes to selection
Ctrl + Left Click + Drag Pan canvas
Mouse Wheel Zoom in/out
Double Right Click Go up one level (exit nested component)
Left Click on Output Port Start creating connection
Drag to Input Port Complete connection
Shift + Left Click on Port Show nested ports
