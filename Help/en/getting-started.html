Getting Started

Creating a New Project

  1. Select File → Create Config from the menu
  2. Choose a location and name for your project
  3. Configure project settings as needed
  4. Click OK to create the project

Adding Components

  1. Open the Components Control window from Window → Components Control
  2. Browse or search for the component you need
  3. Double-click on a component to add it to your model
  4. Components will appear in the Modern Diagram view

Connecting Components

  1. Open the Modern Diagram tab in the Draw Engine window
  2. Click on an output port (blue circle on the right side of a node)
  3. Drag to an input port (green circle on the left side of another node)
  4. Release to create the connection
Tip: Hover over ports to see detailed information in tooltips, including port names, types, and available actions.

Configuring Components

  1. Select a component in the Components Control window
  2. Use the property editor to modify component parameters
  3. Changes are applied immediately

Running Your Model

  1. Use the Calculate menu to control model execution
  2. Click Start to begin execution
  3. Use Pause to temporarily stop execution
  4. Use Reset to reset component states

Monitoring Results

  1. Open Window → Watch Window to create monitoring windows
  2. Add series to watch specific component properties
  3. Charts will update in real-time during execution
