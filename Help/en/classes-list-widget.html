Overview

The Classes List Widget displays a hierarchical list of available component classes that can be added to your neural network model. It's integrated into the Draw Engine Widget and provides easy access to all component types in the system.

Accessing the Widget

The Classes List Widget is displayed in the left panel of the Draw Engine Widget. It's always visible when working with the diagram view.

Features

  • Hierarchical Organization: Component classes organized by category
  • Search Functionality: Search for specific component classes
  • Quick Add: Double-click to add components to your model
  • Category Browsing: Navigate through component categories
  • Integration: Works seamlessly with the diagram view

Usage

Browsing Component Classes

  • Expand category nodes to see component classes within them
  • Navigate through the hierarchy to find the component you need
  • Use search to quickly locate specific components

Adding Components

  1. Find the component class you want to add
  2. Double-click on the component class name
  3. The component will be added to your model
  4. The new component will appear in the diagram view

Searching for Components

Use the search box to find components:

  • Type part of the component class name
  • The list will filter to show matching components
  • Clear the search to show all components again

Component Categories

Components are organized into categories such as:

  • Input components
  • Processing components
  • Output components
  • Neural network layers
  • Computer vision components
  • Motion control components
Tip: The Classes List is synchronized with the diagram view. When you add a component, it appears immediately in the diagram.
