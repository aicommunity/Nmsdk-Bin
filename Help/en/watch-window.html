Overview

The Watch Window is an advanced monitoring tool that allows you to create multiple charts and graphs to observe component behavior in real-time. It provides a flexible interface for monitoring various data types from your neural network model.

Accessing the Widget

The Watch Window can be opened from the Window → Watch Window menu item. Multiple watch windows can be created to monitor different aspects of your model.

Features

  • Multiple Tabs: Create multiple watch tabs, each with its own set of charts
  • Grid Layout: Organize charts in a customizable grid (rows and columns)
  • Multiple Chart Types: Display different types of data visualizations
  • Data Series: Add multiple data series to each chart
  • Real-time Updates: Charts update automatically during model execution
  • Component Property Selection: Choose any component property to monitor
  • Update Interval Control: Configure how often charts refresh

Usage

Creating a Watch Tab

  1. Open the Watch Window
  2. Use the Create Tab action from the menu
  3. A new tab will be created
  4. You can rename tabs by double-clicking on the tab name

Adding Charts

  1. Select a watch tab
  2. Right-click in the tab area
  3. Select Add Chart or use the context menu
  4. A new chart will be added to the grid

Configuring Chart Grid

You can organize charts in a grid layout:

  • Set the number of rows and columns
  • Charts will be arranged in the specified grid
  • Resize charts by dragging splitter handles

Adding Data Series

  1. Click on a chart to select it
  2. Right-click and select Add Series or use the series options menu
  3. Navigate to the component and property you want to monitor
  4. Configure series options (color, name, etc.)
  5. The series will start displaying data

Chart Options

Each chart can be configured with:

  • Custom axis labels and ranges
  • Multiple data series with different colors
  • Update intervals
  • Display modes (continuous, sliding window, etc.)

Series Options

Each data series can be configured with:

  • Custom name and color
  • Data source (component and property)
  • Display style (line, points, etc.)
  • Update frequency

Watch Tab Management

  • Create Tab: Add a new watch tab
  • Close Tab: Close a tab by clicking the X button on the tab
  • Rename Tab: Double-click the tab name to rename it
  • Move Tabs: Drag tabs to reorder them

Update Intervals

You can control how often charts update:

  • Set a global update interval for all charts in a tab
  • Configure individual chart update rates
  • Balance between real-time feedback and performance
Tip: Use different watch tabs for different aspects of your model. For example, one tab for input monitoring, another for output analysis, and a third for internal state observation.

Related Widgets

  • Graph Widget - Simpler single-graph visualization
  • Images Widget - Monitor image data
  • Components List Widget - Select components to monitor
